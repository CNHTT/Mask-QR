## ✨ 功能特点

- 🎨 **个性化定制**：支持自由调整前景/背景颜色、渐变色及码点形状。
- 🖼️ **LOGO 嵌入**：一键上传图片，精准合成带有品牌标识的二维码。
- 📏 **高清导出**：支持 PNG、JPG 以及 **SVG 矢量格式**，完美适配印刷需求。
- 🔒 **隐私保护**：所有生成过程均在浏览器本地完成，不会向服务器上传任何信息。
- 📱 **响应式设计**：完美适配手机、平板及电脑端。

## 🛠️ 技术实现

- **框架**: React / Vite
- **样式**: Tailwind CSS
- **渲染**: HTML5 Canvas & SVG API
