**[AUU QRコード作成器](https://qr.auufree.com/)** は、プライバシー保護と高性能を両立したオンラインQRコード生成ツールです。ビジネスでも個人利用でも、数秒でプロフェッショナルなデザインのQRコードを作成できます。

公式サイト：**[https://qr.auufree.com/](https://qr.auufree.com/)**

---

## ✨ 主な機能

- 🎨 **自由なカスタマイズ**: 前景・背景色、グラデーション、形状などを自由に変更可能。
- 🖼️ **ロゴ挿入**: お手持ちのロゴ画像をアップロードして、QRコード中央に配置できます。
- 📏 **高画質出力**: PNG、JPGに加え、印刷に最適な **SVG（ベクター）形式** に対応。
- 🔒 **プライバシー重視**: 処理はすべてブラウザ内で行われるため、データがサーバーに送信されることはありません。
- 📱 **マルチデバイス対応**: PC、スマホ、タブレットのすべてで快適に利用可能。

## 🛠️ 使用技術

- **フレームワーク**: React / Vite
- **スタイリング**: Tailwind CSS
- **レンダリング**: HTML5 Canvas & SVG API
